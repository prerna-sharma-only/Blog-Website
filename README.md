## 🛠️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/prerna-sharma-only/Blog-Website.git
```

### 2️⃣ Navigate into project

```bash
cd Blog-Website
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the server

```bash
npm start
```

### 5️⃣ Open browser

```bash
http://localhost:3000
```
